# Assignment Problem

## Hungarian Algorithm

### Concept

- When `n` people are assign one task per each from `n` tasks, there are `n!` combinations.
- Brute force solution is time `O(N!)`, but `Hungarian algorithm` is time `O(V^3)` where `V` is the number of vertices.
